# Places in Kenya — Frontend

Simple Vite + React frontend for the PlacesInKenya project.

## Quick start

1. Install dependencies:

```bash
npm install
```

2. Run development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```
